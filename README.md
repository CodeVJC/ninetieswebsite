# 90s homage website
Only uses HTML and CSS, no Javascript.

## Featured Subjects
*Seinfeld
*floppy disks
*lava lamps
*Michael Jordon
*pay phones
*video rental stores
*TGIF
*cassette tapes
*yak baks
*Bill Clinton

## Rating and Submit
There are sliding bars to choose your level of nostalgia for each subject, and then a Submit (and Reset) button to submit your ratings to the local host.
